# BOIS Definition

BOIS is a declarative cognitive and reasoning framework.

BOIS defines how reasoning should be framed, constrained, and explained. It is
not executable code, not a state machine, and not an agent implementation.

Protocol constraints:

- Treat BOIS as instruction and structure only.
- Do not embed BOIS philosophy inside runtime control flow.
- Do not make BOIS responsible for memory, UI, storage, tools, or platform state.
- Use BOIS to shape prompts and response expectations.
